# 🎲 Dice Roller

A clean, interactive dice roller web app — roll any standard tabletop dice (d4, d6, d8, d10, d12, d20, d100) with real-time results and roll history.

**Live site:** https://yuanfengli168.github.io/dice-roller/

## Features

- Roll 1–20 dice of any standard type
- Animated results with min/max/avg stats
- Full roll history (last 50 rolls)
- Keyboard shortcut: press **Space** or **Enter** to roll
- Dark-mode UI, fully responsive

## Stack

Pure HTML + CSS + JavaScript — zero dependencies, zero build step.

## Local dev

```bash
# Just open index.html in your browser, or:
npx serve .
```

## Deploy

GitHub Actions deploys to GitHub Pages on every push to `main`.
